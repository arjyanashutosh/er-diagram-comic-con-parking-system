# Event Parking Management System – ER Diagram

## About
This project models how a parking system manages vehicles, parking spots, zones, sessions, tickets, and payments in a structured and scalable way.

## Key Features
- Vehicles are categorized (car, bike, EV, etc.)  
- Parking spots are organized by zones and levels  
- Spots can be reserved for categories like VIP, staff, or EV  
- One vehicle can have multiple parking sessions  
- Entry and exit timestamps are tracked for each session  
- Tickets are generated for each parking session  
- Payments are linked to parking sessions  
- Parking spots can be reused across multiple sessions
